
Petit guide pour ajouter ses modifications à une branche via le terminal:

1
git status = Verifie que tous les fichiers locaux sont mis à jour sur le git

2
git add *|[nomFichier1] [...] = ajoute tous les fichiers sur git

3
git commit -m "[message]" = envoie les fichiers pour confirmation

4
git checkout [nomBranche] = se déplacer dans la bonne branche pour faire un push

5
git push = Confirmer les modifications et ecrasements des fichiers de la branche
